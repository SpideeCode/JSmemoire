# 🧠 Jeu de Mémoire

Petit jeu de mémoire réalisé en **HTML**, **CSS** et **JavaScript**.

L’objectif est de retrouver les paires d’emojis identiques le plus rapidement possible.  
Un chrono démarre dès le premier clic et le nombre de coups est comptabilisé.

## ⚙️ Fonctionnalités
- Grille de cartes interactives
- Mélange aléatoire des emojis
- Compteur de coups et minuterie
- Bouton pour recommencer une partie

## ▶️ Lancer le jeu
Ouvrir le fichier `index.html` dans un navigateur.
